## 二.功能

書籍搜索，書籍預覽

## 三.技術棧

（react18）React Hooks

（react庫）React Router6，React icon 

（Http請求交互）fetch & axios

## 四.項目亮點

為了減少請求，為搜索功能增加防抖優化，為图片增加懶加載，以及創建 resize 函數防抖（自定義debounce & lazy-load Hooks）
